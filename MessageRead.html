Unprotected
Unprotected, Not using any encryption method.
Encrypt Via Secure Portal
Secure Portal, the most secure method of delivery, requires the recipient to authenticate with a password or two-factor before being able to access the encrypted message.
Encrypt via Easy-Secure
Easy-Secure, embeds an access-key in a unique link sent to the recipient that provides "one-click" access to the encrypted message.
Encrypt with TLS Verify
TLS Verify, utilizes only secure versions of the Transport Layer Security protocol to scure a message while in route to its destination. The recipient does not have to take additional steps to view the secure email.
